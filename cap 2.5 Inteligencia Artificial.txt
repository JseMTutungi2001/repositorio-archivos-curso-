[Machine Learning:]

Programas que resuelven problemas -> programas que aprenden a resolver problemas

esto quiere decir que el machine learning frente a un desarrollador realiza un 
programa o un algoritmo que resuelve problemas, el machine learning ofrece 
algoritmos que aprenden a partir de datos y ejemplos dados, y son ellos mismos
los que configuran un algoritmo apra resolver un problema.

de esta forma en el machine learning nos encontramos hay 3 areas especiales de 
desarrollo:

(1) aprendizaje supervisado:
en que un sistema es capaz de aprender a partir de  una serie de datos etiquetados
con unos resultados, ya dados, es capaz de hacer predicciones o de emular ese
comportamiento y darnos resultados.

(2) aprendizaje no supervisado:
en el cual el sistema es capaz de reconocer patrones, reconocer comportamientos,
a partir de un datasend dado y de esta manera acada ves que hay una entrada
nueva puede clasificarla dentro de ese algoritmo y darnos un resultado u 
informacion sobre esa entrada.

(3) aprendizaje por refuerzo:
en el cual el sistema va a ir dando resultados y espera un positivo o un negativo
pòr parte del entrenador y esto lepermite ir reforzando el algoritmo en unos 
aspectos o en otros.

 *[Redes Neuronales]
aveces los sistemas en los que se aplica el machine learning son demasiado complejos
tienen demasiadas variables y requieren del uso de redes neuronales.

Las redes neuronales lo que hace ees crear una primera capa de variables sencillas
y de entre la interaccion de las variables sencillas va creando distintas capaz
en la que va dedusiendo nuevas variables que van interaccionando de una capa a 
otra de forma que van reforzando que el algoritmo se comporte de una manera 
determinada hasta llegar a unresultado de una forma mas eficiente que si tuvieramos
que usar sistemas con todas las variables dentro del modelo matematico completo.
